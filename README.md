# Skill30 Framework

> A Professional Prompt Framework for Building High-Quality 30-Day Skill Learning Plans.

Skill30 Framework 是一个用于生成高质量30天技能成长计划的 Prompt Framework。

它不是简单的学习清单生成器，而是一套基于：

- 刻意练习理论（Deliberate Practice）
- 技能拆解（Skill Decomposition）
- 知识依赖（Knowledge Dependency）
- 用户画像适配（User Persona Adaptation）
- 真实性验证（Authenticity Validation）

构建的技能规划系统。

---

# Features

## Skill Decomposition

将宽泛技能拆解为可训练的能力单元。

例如：

Python

↓

变量

↓

流程控制

↓

函数

↓

项目实践


---

## Learning Pattern Recognition

根据技能类型自动调整学习方式：

- 编程型
- 语言型
- 创作型
- 动作型
- 知识型
- 工具型


---

## User Persona Adaptation

根据学习者类型调整：

- 任务密度
- 容错设计
- 反馈方式
- 语言风格


支持：

- 冲动型
- 完美主义型
- 逃避型
- 补偿型
- 功利型
- 体验型


---

## Authenticity Gate

禁止：

- 虚构软件功能
- 编造按钮路径
- 围绕错误前提生成方案
